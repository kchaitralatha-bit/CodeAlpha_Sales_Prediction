# Sales Prediction using Python

## 📌 Project Overview
This project predicts product sales using Machine Learning based on advertising expenditure across different media such as TV, Radio, and Newspaper.

## 🎯 Objective
To analyze advertising data and build a regression model capable of predicting future sales.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## 📂 Project Workflow
1. Import Libraries
2. Load Dataset
3. Data Cleaning
4. Exploratory Data Analysis (EDA)
5. Feature Selection
6. Train-Test Split
7. Model Training
8. Model Evaluation
9. Sales Prediction

## 📊 Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

## 🚀 How to Run
1. Clone the repository.
2. Install the required libraries.
3. Open the notebook.
4. Run all cells.
